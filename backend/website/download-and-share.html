We think these videos are so valuable that they should be always be available for anyone and from anywhere, and that is why we have uploaded them to IPFS.

To use the download links that start with ipfs:// or ipns:// you need an app. You can also manually run an IPFS node and use a browser extension such as this one.

Once you have an IPFS node, you can download all the videos by pinning the following directory to your node.

ipfs pin /ipns/resources.spiritualityresources.net

This way you will also help the availability of the videos, so if a country decides to block YouTube, or if YouTube itself decides to remove the video from their servers or goes out of business, the videos will still be available from IPFS.

You can view a list of all the videos that are there by clicking here.

If you only want to download all the videos, you would run :

ipfs get /ipns/resources.spiritualityresources.net

For more information on IPFS, visit the IPFS homepage.
